# Финансовый трекер (Finance Control)

## Автор
Орлов Тимур Юрьевич, группа ИСП(9)-23-1

## Описание
Приложение для учёта личных финансов пользователя с возможностью добавления доходов и расходов, установки лимитов и защиты данных PIN-кодом.

## Технологии
